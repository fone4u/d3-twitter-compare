About

This visualisation shows a comparison of the Follow @SCOREProject and Follow @ukoer Twitter following. The analysis was conducted based on data collected on the 1st May 2012 (the data used excludes protected Twitter accounts). The data was extracted via the Twitter API using Google Spreadsheets and the community of followers analysed using R with the iGraph package. Each node represents a single follower, the size of the node being scaled using an individuals Betweenness Centrality (a measure of the community bridging capability), In-Degree (how many other people who follower SCOREProject or ukoer also follow the person represented by the node) and Followers count (how many people follower the person represented by the node. As part of the analysis community structure detection algorithm proposed Joerg Reichardt and Stefan Bornholdt (Spin Glass) was used to identify five groups within the following of SCOREProject and ukoer. These groups are colour coded and word clouds for each of these subcommunities was generated based on Twitter Profile descriptions.
